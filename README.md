# forge-nix

> Foundry git submodule / nix dependency manager 


## Install Methods

## `via local repo`
```bash
git clone https://github.com/sambacha/forge-nix/ && cd forge-nix && bash installer
```

## `via url`

```shell
curl -L https://raw.githubusercontent.com/sambacha/forge-nix/master/installer | bash
```

## usage

```shell
$ forge2nix --help
```
